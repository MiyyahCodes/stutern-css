Margin: The area surrounding an element. On a page, margins can be used to move an element left, right, up, and down. The margin has no background color and is entirely transparent. The area surrounding the element is made evident. You can adjust the margin size for each side of the element separately. The margin pushes adjacent items away, resulting in the gap.

Padding: The distance between an element and the relevant content contained therein. It controls how components appear and are positioned inside a container. It also displays the backdrop of the container that the element is inside of. Background colors can alter padding since they clear the space surrounding the content. It either increases the size of the element or decreases the information inside to generate the gap. The element's size enlarges by default.

Border: The border property is a shorthand syntax in CSS that accepts multiple values for drawing a line around the element it is applied to.
